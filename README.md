# IntimoMare

IntimoMare is a Ruby on Rails web application designed to showcase and manage collections of intimate apparel, swimwear, and beachwear. Inspired by Mediterranean elegance and effortless coastal fashion.

## Features

- 🌟 Display curated clothing collections with imagery and descriptions
- 📦 Inventory and stock management
- 🛒 Shopping cart and secure checkout integration
- 🧾 Admin panel for product editing and analytics
- 🌐 Multilingual support (EN/IT/FR)

## Tech Stack

- **Ruby on Rails 7**
- **PostgreSQL** for data storage
- **Hotwire / Turbo** for seamless interactivity
- **Tailwind CSS** for styling
- **Devise** for user authentication
